# Nulltrack — Issues & Feedback

This is the public issue tracker for **[Nulltrack](https://nulltrack.app)**, the mobile tracker and DAW for iPhone.

## Found a bug? Want a feature?

1. **[Search existing issues](https://github.com/nulltrack/issues-feedback/issues?q=is%3Aissue)** first — someone may have already reported it.
2. If it exists, add a 👍 reaction to the issue instead of opening a new one.
3. If it's new, **[open an issue](https://github.com/nulltrack/issues-feedback/issues/new/choose)** — there are templates for bug reports and feature requests.

**[Most-requested features →](https://github.com/nulltrack/issues-feedback/issues?q=is%3Aissue+is%3Aopen+label%3Afeature-request+sort%3Areactions-%2B1-desc)**
